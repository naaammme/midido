## 简介

MidiDo是一款使用PyQt6开发的MIDI自动演奏辅助工具，可以在游戏中自动演奏MIDI音乐。

## 功能

- 目前仅支持网易明日之后pc端的钢琴、竖琴、吉他等乐器(因为我只玩过这个)
-  在线音乐库，一键下载收藏


## 安装
[点击下载](https://github.com/naaammme/midido/releases)

## 使用说明

1. 在"设置"页面选择游戏窗口和乐器配置
2. 在"我的收藏"中选择 MIDI 文件
3. 调整播放参数（速度、音高、延迟）
4. 点击"开始"或按 F1 开始演奏
5. 游戏窗口内按 F1 同样可以开始弹奏,再按一次停止弹奏
